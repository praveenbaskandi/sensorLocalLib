# @praveen/sensors-bridge

A React Native native module providing location and gyroscope sensors using core Android and iOS APIs.

## Installation


```sh
npm install @praveen/sensors-bridge
```


## Usage


```js
import { multiply } from '@praveen/sensors-bridge';

// ...

const result = multiply(3, 7);
```
